# Heart Disease Analysis using Tableau

## Overview
This project analyzes heart disease data to identify key risk factors such as age, smoking habits, and lifestyle choices. It uses Tableau for visualization and a simple Flask app to display basic insights.

## Dataset
The dataset contains 4500 records with 18 features including:
- AgeCategory
- BMI
- Smoking
- PhysicalActivity
- HeartDisease
- SleepTime
- and more...

## Tools & Technologies
- Tableau (Data Visualization)
- Python (Pandas, NumPy)
- Flask (Web Framework)

## Features
- Interactive Tableau dashboard
- Analysis of heart disease distribution
- Age-wise risk analysis
- Impact of smoking on heart disease
- Basic web dashboard using Flask
